题目：燃气调压器故障预测与诊断

小区燃气调压器是确保供气安全和稳定的重要设备。然而，常见故障如过滤器堵塞、阀座损坏和隔膜损坏可能会影响其正常运行。当燃气调压器出现故障时，通常会表现为以下三种常见故障：过滤器堵塞、阀座损坏和隔膜损坏。这些故障可能会导致燃气调压器无法正常工作，进而影响到燃气供应系统的安全性和稳定性。
过滤器堵塞的主要原因是燃气中含有的杂质和污染物，在长时间使用后，这些杂质和污染物会积聚在过滤器中，导致通道堵塞。其表现形式为燃气通过的流量减小，从而使得输出压力无法正常调节，可能出现输出压力波动、不稳定甚至无法维持设定值的情况。
阀座损坏通常是由于长期使用或不当操作导致的阀门密封面磨损或变形，使得阀门无法完全关闭或密封不严。阀座损坏会导致燃气泄漏或无法完全关闭，从而造成输出压力偏高或无法稳定调节的情况，甚至可能引发安全隐患。
隔膜损坏通常是由于材料老化、疲劳、物理损伤或操作不当等因素引起的，使得隔膜的密封性能降低或失效。隔膜损坏会导致燃气调压器的密封性能下降，可能会引起燃气泄漏、输出压力波动大或无法维持稳定的情况，严重时可能会导致系统压力失控和安全隐患。
为了提前预防故障并进行及时维修，我们希望通过实时的燃气压力监测数据来预测可能出现的故障。我们提供了实时的燃气压力监测数据，包括输入压力（in_pressure）、输出压力（out_pressure）以及环境温度（temperature）。数据以每分钟为单位记录。基于提供的实时监测数据，请完成以下任务。

一、对输入压力、输出压力和环境温度的数据进行可视化分析，探索它们之间的关系以及可能的异常情况。
二、故障预测模型建立。基于监测数据，建立预测模型，以预测燃气调压器可能出现的故障情况（过滤器堵塞、阀座损坏、隔膜损坏），并评估模型的准确性和鲁棒性。
三、故障诊断与维修策略：提出针对不同故障类型的预防和诊断策略，以及可能的维修措施。

要求：提交报告，包括数据可视化、模型建立和评估结果，并附带详细的解释和推理过程。
报告格式不限，但应清晰展示分析过程和结论。 评分将考虑报告的逻辑性、模型预测准确性、提出的策略可行性等因素。

异常情况分析

这三种故障（过滤器堵塞、阀座损坏和隔膜损坏）在出口压力和进口压力上的表现有明显的区别。以下是对每种故障的分析与假设：

过滤器堵塞

表现形式：
  • 燃气流量减小：由于杂质和污染物积聚在过滤器中，导致燃气流量减少。
  • 出口压力无法正常调节：过滤器堵塞会使出口压力波动、不稳定，甚至无法维持设定值。
出口压力：
  • 降低：由于燃气流量减少，出口压力通常会降低。
  • 波动和不稳定：堵塞可能导致压力调节不稳定，出口压力波动较大。
进口压力：
  • 基本不变：进口压力通常不会受到过滤器堵塞的直接影响。
  • 轻微升高（极端情况下）：如果过滤器非常严重地堵塞，进口压力可能会略微升高，但这种情况较少见。

阀座损坏

表现形式：
  • 密封不严或无法完全关闭：阀门密封面磨损或变形，使得阀门无法完全关闭或密封不严。
  • 燃气泄漏：导致燃气泄漏或无法完全关闭。
出口压力：
  • 偏高：如果阀座损坏导致燃气泄漏，出口压力可能会偏高，因为燃气没有完全被调节到预期水平。
  • 不稳定：阀座损坏会使出口压力不稳定，调节困难，波动较大。
进口压力：
  • 基本不变：进口压力通常不受阀座损坏的直接影响。

隔膜损坏

表现形式：
  • 密封性能降低或失效：由于材料老化、疲劳或物理损伤，隔膜的密封性能下降。
  • 调压失效：隔膜损坏会导致燃气调压器的调压功能失效。
出口压力：
  • 波动大、不稳定：隔膜损坏会导致出口压力波动较大，无法稳定调节。
  • 无法维持设定值：严重时，出口压力无法维持在设定值，可能偏高或偏低。
进口压力：
  • 基本不变：进口压力通常不受隔膜损坏的直接影响。

总结

故障类型 出口压力 进口压力
过滤器堵塞 降低或波动、不稳定 基本不变，可能轻微升高
阀座损坏 偏高或不稳定 基本不变
隔膜损坏 波动大、不稳定，无法维持设定值 基本不变

如何监测和检测这些故障

  • 过滤器堵塞：监测出口压力的下降和波动情况，结合流量监测（燃气流量减小）。
  • 阀座损坏：检查出口压力偏高或不稳定的情况，并进行泄漏检测。
  • 隔膜损坏：观察出口压力的剧烈波动和无法维持设定值的情况，检查调压功能是否失效。

问题与解决

  • 由于我们没有故障标签，所以我们需要做一个聚类来检测哪些属于故障

  • 由于只按一户人家一个样本的话样本量太少，容易欠拟合。

  • 如果按每一分钟作为一个样本缺乏时间尺度的信息，比如波动无法体现，且一天中不同的使用情况也会产生不同的输入压力与输出压力，所以短时间的信息缺乏时间语义。

  • 于是，我决定使用一户人家一天的数据作为一个样本数据，其中设置的统计量特征主要有以下几个：进出口压力的最大值、最小值、方差、偏度系数、峰度系数。

  • 由于特征过多，需要pca降维之后才能进行聚类

  • 我根据这些特征进行k-means聚类，分为两类，将故障天数标出，然后去检查这些天数，对他们进行标记，标记为三种故障。

  • 只有一个被分成故障聚类效果还是不行

  • 由于第一与第三种异常题目给出的特点基本一致，如何将两者区分开来，我们做出假设，由于第三种为泄露，我们认为如果进口与出口压力差值较大，那么就认为是第一种。标签手动标记

  • 然后在有标签后，用随机森林对以上数据建立模型，并测试。与单决策树进行比较。

def calc_stats(group):
	stats = {}
	for col in ['Pressure_Difference', 'Out_Pressure (kPa)']:
		stats[f'{col}_max'] = group[col].max()
		stats[f'{col}_min'] = group[col].min()
		stats[f'{col}_var'] = group[col].var()
		stats[f'{col}_skew'] = skew(group[col])
		stats[f'{col}_kurtosis'] = kurtosis(group[col])
	return pd.Series(stats)
