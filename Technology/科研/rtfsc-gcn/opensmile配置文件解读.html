OpenSmile配置文件解读

这个 openSMILE 配置文件定义了从音频信号中提取各种特征的过程。下面是对配置文件各个部分的详细解释：

文件头部注释

///////////////////////////////////////////////////////////////////////////
// openSMILE configuration template file generated by SMILExtract binary //
// you must manually adapt this file and change at least the             //
// 'reader/writer.dmLevel =' lines.                                      //
///////////////////////////////////////////////////////////////////////////
  • 这是一个由 SMILExtract 生成的 openSMILE 配置文件模板。
  • 用户需要手动调整文件，特别是 reader/writer.dmLevel 行。

组件管理器配置

;===== component manager configuration (= list of enabled components!) =====

[componentInstances:cComponentManager]
instance[dataMemory].type = cDataMemory
instance[waveSource].type = cWaveSource
instance[framer].type = cFramer
instance[mzcr].type=cMZcr
instance[csvSink].type = cCsvSink
instance[pe].type=cVectorPreemphasis
instance[win].type=cWindower
instance[fft].type=cTransformFFT
instance[fftmag].type=cFFTmagphase
instance[melspec].type=cMelspec
instance[mfcc].type=cMfcc
instance[acf].type=cAcf
instance[cepstrum].type=cAcf
instance[pitchACF].type=cPitchACF
instance[sma].type=cContourSmoother
instance[delta].type=cDeltaRegression
instance[energy].type = cEnergy
  • 定义了启用的组件及其类型。
  • cDataMemory：默认数据内存组件。
  • cWaveSource：读取音频文件。
  • cFramer：将音频信号分割成帧。
  • cMZcr：计算过零率（Zero Crossing Rate）。
  • cCsvSink：将结果输出到 CSV 文件。
  • cVectorPreemphasis：对音频信号进行预加重处理。
  • cWindower：应用窗口函数。
  • cTransformFFT：进行快速傅里叶变换（FFT）。
  • cFFTmagphase：计算 FFT 的幅度和相位。
  • cMelspec：计算梅尔频谱。
  • cMfcc：计算 MFCC 特征。
  • cAcf：计算自相关函数（用于基音检测）。
  • cPitchACF：基于自相关函数的基音检测。
  • cContourSmoother：平滑轮廓特征。
  • cDeltaRegression：计算特征的一阶差分。
  • cEnergy：计算音频信号的能量。

线程设置

; run single threaded (nThreads=1)
; NOTE: a single thread is more efficient for processing small files, since multi-threaded processing involves more 
;       overhead during startup, which will make the system slower in the end
nThreads=1
; do not show any internal dataMemory level settings 
; (if you want to see them set the value to 1, 2, 3, or 4, depending on the amount of detail you wish)
printLevelStats=0
  • 设置为单线程运行 (nThreads=1)，对于小文件更高效。
  • printLevelStats=0：不显示内部数据内存级别的设置。

组件配置

cWaveSource 组件

[waveSource:cWaveSource]
writer.dmLevel = wave
basePeriod = -1
filename = \cm[inputfile(I):file name of the input wave file]
properTimestamps = 0
monoMixdown = 1
start = 0
end = -1
endrel = 0
noHeader = 0
outFieldName = pcm
  • 读取音频文件。
  • writer.dmLevel = wave：写入的数据级别为 wave。
  • basePeriod = -1：基础周期设置为 -1，表示使用整个文件。
  • filename：输入音频文件名。
  • properTimestamps = 0：不使用精确时间戳。
  • monoMixdown = 1：将立体声混合为单声道。
  • start 和 end：开始和结束时间点，默认处理整个文件。
  • outFieldName = pcm：输出字段名为 pcm。

cFramer 组件

[framer:cFramer]
reader.dmLevel = wave
writer.dmLevel = waveframes
copyInputName = 1
EOIlevel = 0
allowLastFrameIncomplete = 0
frameMode = fixed
frameSize = 0.025000
frameStep = 0.010000
frameCenterSpecial = left
noPostEOIprocessing = 1
  • 将音频信号分割成帧。
  • reader.dmLevel = wave：从 wave 数据级别读取。
  • writer.dmLevel = waveframes：写入的数据级别为 waveframes。
  • frameSize = 0.025000：每帧大小为 25 毫秒。
  • frameStep = 0.010000：帧移为 10 毫秒。

cEnergy 组件

[energy:cEnergy]
reader.dmLevel = waveframes
writer.dmLevel = energy
nameAppend = energy
copyInputName = 1
EOIlevel = 0
processArrayFields = 0
includeSingleElementFields = 0
preserveFieldNames = 1
htkcompatible = 0
rms = 0
energy2 = 0
log = 1
escaleLog = 1
escaleRms = 1
escaleSquare = 1
ebiasLog = 0
ebiasRms = 0
ebiasSquare = 0
  • 计算音频信号的能量。
  • reader.dmLevel = waveframes：从 waveframes 数据级别读取。
  • writer.dmLevel = energy：写入的数据级别为 energy。
  • log = 1：对能量取对数。
  • escaleLog = 1：对数能量的缩放。

cVectorPreemphasis 组件

[pe:cVectorPreemphasis]
reader.dmLevel=waveframes
writer.dmLevel=framespe
k = 0.97
de = 0
  • 对音频信号进行预加重处理。
  • reader.dmLevel=waveframes：从 waveframes 数据级别读取。
  • writer.dmLevel=framespe：写入的数据级别为 framespe。
  • k = 0.97：预加重系数。

cWindower 组件

[win:cWindower]
reader.dmLevel=framespe
writer.dmLevel=winframes
copyInputName = 1
processArrayFields = 1
winFunc = ham
gain = 1.0
offset = 0
  • 应用窗口函数。
  • reader.dmLevel=framespe：从 framespe 数据级别读取。
  • writer.dmLevel=winframes：写入的数据级别为 winframes。
  • winFunc = ham：使用汉明窗。

cTransformFFT 组件

[fft:cTransformFFT]
reader.dmLevel=winframes
writer.dmLevel=fft
copyInputName = 1
processArrayFields = 1
inverse = 0
zeroPadSymmetric = 0
  • 进行快速傅里叶变换。
  • reader.dmLevel=winframes：从 winframes 数据级别读取。
  • writer.dmLevel=fft：写入的数据级别为 fft。

cFFTmagphase 组件

[fftmag:cFFTmagphase]
reader.dmLevel=fft
writer.dmLevel=fftmag
copyInputName = 1
processArrayFields = 1
inverse = 0
magnitude = 1
phase = 0
  • 计算 FFT 的幅度。
  • reader.dmLevel=fft：从 fft 数据级别读取。
  • writer.dmLevel=fftmag：写入的数据级别为 fftmag。
  • magnitude = 1：计算幅度。
  • phase = 0：不计算相位。

cMelspec 组件

[melspec:cMelspec]
reader.dmLevel=fftmag
writer.dmLevel=melspec
copyInputName = 1
processArrayFields = 1
htkcompatible = 1
nBands = 26
usePower = 1
lofreq = 0
hifreq = 8000
specScale = mel
inverse = 0
  • 计算梅尔频谱。
  • reader.dmLevel=fftmag：从 fftmag 数据级别读取。
  • writer.dmLevel=melspec：写入的数据级别为 melspec。
  • nBands = 26：梅尔滤波器的数量。
  • lofreq = 0 和 hifreq = 8000：频率范围。
  • specScale = mel：使用梅尔刻度。

cMfcc 组件

[mfcc:cMfcc]
reader.dmLevel=melspec
writer.dmLevel=ft0
copyInputName = 1
processArrayFields = 1
firstMfcc = 0
lastMfcc  = 12
cepLifter = 22.0
htkcompatible = 1
  • 计算 MFCC 特征。
  • reader.dmLevel=melspec：从 melspec 数据级别读取。
  • writer.dmLevel=ft0：写入的数据级别为 ft0。
  • firstMfcc = 0 和 lastMfcc = 12：计算 13 个 MFCC 系数。
  • cepLifter = 22.0：倒谱提升系数。

cMZcr 组件

[mzcr:cMZcr]
reader.dmLevel=waveframes
writer.dmLevel=mzcr
copyInputName = 1
processArrayFields = 1
zcr = 1
amax = 0
mcr = 0
maxmin = 0
dc = 0
  • 计算过零率。
  • reader.dmLevel=waveframes：从 waveframes 数据级别读取。
  • writer.dmLevel=mzcr：写入的数据级别为 mzcr。
  • zcr = 1：计算过零率。

cAcf 和 cCepstrum 组件

[acf:cAcf]
reader.dmLevel=fftmag
writer.dmLevel=acf
nameAppend = acf
copyInputName = 1
processArrayFields = 1
usePower = 1
cepstrum = 0

[cepstrum:cAcf]
reader.dmLevel=fftmag
writer.dmLevel=cepstrum
nameAppend = acf
copyInputName = 1
processArrayFields = 1
usePower = 1
cepstrum = 1
  • 计算自相关函数和倒谱。
  • reader.dmLevel=fftmag：从 fftmag 数据级别读取。
  • writer.dmLevel=acf 和 writer.dmLevel=cepstrum：分别写入 acf 和 cepstrum 数据级别。
  • cepstrum = 0 和 cepstrum = 1：是否计算倒谱。

cPitchACF 组件

[pitchACF:cPitchACF]
reader.dmLevel=acf;cepstrum
writer.dmLevel=pitch
copyInputName = 1
processArrayFields=0
maxPitch = 500
voiceProb = 1
voiceQual = 0
HNR = 0
F0 = 1
F0raw = 0
F0env = 0
voicingCutoff = 0.550000
  • 基于自相关函数的基音检测。
  • reader.dmLevel=acf;cepstrum：从 acf 和 cepstrum 数据级别读取。
  • writer.dmLevel=pitch：写入的数据级别为 pitch。
  • maxPitch = 500：最大基音频率。
  • F0 = 1：计算基音频率。

cContourSmoother 组件

[sma:cContourSmoother]
reader.dmLevel=ft0;mzcr;pitch;energy
writer.dmLevel=lld
nameAppend = sma
copyInputName = 1
noPostEOIprocessing = 0
smaWin = 3
  • 平滑轮廓特征。
  • reader.dmLevel=ft0;mzcr;pitch;energy：从多个数据级别读取。
  • writer.dmLevel=lld：写入的数据级别为 lld。
  • smaWin = 3：平滑窗口大小为 3。

cDeltaRegression 组件

[delta:cDeltaRegression]
reader.dmLevel=lld
writer.dmLevel=lld_de
nameAppend = de
copyInputName = 1
noPostEOIprocessing = 0
deltawin=2
blocksize=1
  • 计算特征的一阶差分。
  • reader.dmLevel=lld：从 lld 数据级别读取。
  • writer.dmLevel=lld_de：写入的数据级别为 lld_de。
  • deltawin=2：差分窗口大小为 2。

数据输出配置

[csvSink:cCsvSink]
reader.dmLevel = ft0;mzcr;pitch;lld_de
errorOnNoOutput = 0
filename = \cm[outputfile(O):file name of the output CSV file]
delimChar = ;
append = 0
timestamp = 1
frameTime = 1
number = 1
frameIndex = 1
printHeader = 1
flush = 0
  • 将结果输出到 CSV 文件。
  • reader.dmLevel = ft0;mzcr;pitch;lld_de：从多个数据级别读取。
  • filename：输出 CSV 文件名。
  • delimChar = ;：CSV 分隔符为分号。
  • timestamp = 1 和 frameTime = 1：包含时间戳和帧时间。
  • printHeader = 1：输出文件包含表头。

文件结尾

// ################### END OF openSMILE CONFIG FILE ######################
  • 标记配置文件的结束。

通过这个配置文件，openSMILE 可以从音频文件中提取多种特征，并将结果输出到 CSV 文件中。每个组件都有其特定的功能，如读取音频、分帧、预加重、窗口化、FFT、梅尔频谱、MFCC、过零率、自相关函数、基音检测、平滑等。
